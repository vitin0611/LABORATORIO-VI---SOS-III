# LABORATORIO-VI---SOS-III
Este repositorio contiene el desarrollo de cuatro prácticas enfocadas en la administración y seguridad de sistemas Linux utilizando Red Hat Enterprise Linux 10 (RHEL 10). Durante estas prácticas se implementan técnicas de cifrado de archivos, administración de firewall, detección de intrusos y autenticación multifactor para fortalecer la seguridad.
